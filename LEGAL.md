# Notes legales

Ce depot separe volontairement deux choses :

- le code du patcher statique et les outils, places sous licence MIT dans `LICENSE-CODE.md` ;
- les textes de traduction, qui sont une oeuvre derivee du texte original du jeu.

Les fichiers dans `assets/translations/` contiennent une traduction francaise complete de textes issus de *Esoteric Ebb*. Avant une publication publique, il est recommande d'obtenir l'accord du createur, de l'editeur, ou de publier selon les regles de la plateforme de mods choisie.

Le patcher statique ne doit pas distribuer d'assets Unity modifies. Il applique les remplacements uniquement sur l'installation locale du joueur, a partir des fichiers de traduction fournis avec le patch.

En attendant une autorisation explicite, le choix le plus propre est de garder le depot prive, ou de publier uniquement le code et les scripts en demandant aux utilisateurs de generer localement les fichiers de traduction depuis une source autorisee.
